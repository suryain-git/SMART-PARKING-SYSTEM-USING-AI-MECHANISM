# Smart Parking System Using AI Mechanism

## Overview

A real-time AI-powered parking management system that detects occupied and vacant parking slots using YOLOv8, OpenCV, and Flask.

The system processes video streams, identifies vehicles, and automatically determines parking slot availability.

## Features

- Real-time vehicle detection
- Occupied parking slot detection
- Vacant parking slot identification
- Live video processing
- Parking occupancy monitoring

## Technologies Used

- Python
- Flask
- OpenCV
- YOLOv8
- NumPy
- HTML
- CSS
- JavaScript
